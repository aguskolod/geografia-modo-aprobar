# Geografía | Modo aprobar

Curso intensivo de Geografía turística basado en el resumen de estudio incluido en el repositorio.

Incluye:

- 11 lecciones organizadas por región.
- Material fuente integrado dentro de cada lección.
- 20 fichas de memoria activa.
- 49 preguntas con corrección automática.
- Simulacros de 20 preguntas y nota sobre 10.
- Registro local de progreso y errores.

El material fue compartido para esta aplicación con autorización de su titular.
